# Keyboard Shortcuts

Quick reference for all keyboard shortcuts in Isaac Shader Studio.

## Main Actions

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Alt+N` | New Shader | Create a new blank shader |
| `Ctrl+Enter` | Compile & Run | Compile shader and apply to preview |
| `Ctrl+S` | Save Shader | Save to browser localStorage |
| `Ctrl+E` | Export Shader | Open export dialog |
| `Ctrl+I` | Import Shader | Open import dialog |

## Why Alt+N and Ctrl+Enter?

- `Ctrl+N` opens a new browser window
- `Ctrl+R` reloads the page
- `Ctrl+Shift+N` opens incognito mode

These shortcuts were chosen to avoid conflicts with browser defaults.

## Editor Shortcuts

The shader editors use Ace Editor with standard shortcuts:

### Basic Editing

| Shortcut | Action |
|----------|--------|
| `Ctrl+Z` | Undo |
| `Ctrl+Y` or `Ctrl+Shift+Z` | Redo |
| `Ctrl+X` | Cut line/selection |
| `Ctrl+C` | Copy line/selection |
| `Ctrl+V` | Paste |
| `Ctrl+D` | Duplicate line |
| `Ctrl+Shift+K` | Delete line |

### Find & Replace

| Shortcut | Action |
|----------|--------|
| `Ctrl+F` | Find |
| `Ctrl+H` | Find & Replace |
| `F3` or `Ctrl+G` | Find next |
| `Shift+F3` | Find previous |

### Comments

| Shortcut | Action |
|----------|--------|
| `Ctrl+/` | Toggle line comment |
| `Ctrl+Shift+/` | Toggle block comment |

### Line Operations

| Shortcut | Action |
|----------|--------|
| `Alt+Up` | Move line up |
| `Alt+Down` | Move line down |
| `Ctrl+Shift+D` | Duplicate line |
| `Ctrl+Shift+K` | Delete line |

### Selection

| Shortcut | Action |
|----------|--------|
| `Ctrl+A` | Select all |
| `Ctrl+L` | Select line |
| `Ctrl+D` | Select next occurrence |
| `Ctrl+Shift+L` | Select all occurrences |
| `Shift+Alt+Up/Down` | Add cursor above/below |

### Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl+G` | Go to line |
| `Ctrl+Home` | Go to start of file |
| `Ctrl+End` | Go to end of file |
| `Ctrl+Left` | Move word left |
| `Ctrl+Right` | Move word right |
| `Ctrl+Up` | Scroll up (no cursor move) |
| `Ctrl+Down` | Scroll down (no cursor move) |

### Code Folding

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+[` | Fold selection |
| `Ctrl+Shift+]` | Unfold selection |
| `Ctrl+K Ctrl+0` | Fold all |
| `Ctrl+K Ctrl+J` | Unfold all |

## Game Preview Controls

When the game preview is focused:

| Key | Action |
|-----|--------|
| `W` / `Up` | Move player up |
| `A` / `Left` | Move player left |
| `S` / `Down` | Move player down |
| `D` / `Right` | Move player right |
| `Space` | Use active item |

## Platform Notes

### macOS

Replace `Ctrl` with `Cmd` for most shortcuts:
- `Cmd+S` to save
- `Cmd+Enter` to compile
- `Alt+N` stays the same (Option+N)

### Linux

Shortcuts work the same as Windows with `Ctrl`.

## Tips

- **Compile often**: Use `Ctrl+Enter` frequently to catch errors early
- **Quick save**: `Ctrl+S` saves instantly without confirmation
- **Error navigation**: Click error messages in the console to jump to the line
- **Multiple cursors**: Hold `Alt` and click to add cursors for multi-editing
